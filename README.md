# 🎨 Electron Drawing Overlay

일렉트론을 이용한 투명 화면 그리기 도구입니다.

## 주요 기능
- **그리기 모드**: 화면 위에 자유롭게 낙서 (마우스 잡기)
- **마우스 모드**: 투과 모드를 통해 바탕화면이나 다른 앱 클릭 가능
- **ESC**: 화면 전체 지우기
- **색상 변경**: Red, Blue, Green, Black 지원
